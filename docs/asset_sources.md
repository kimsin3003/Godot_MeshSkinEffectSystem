# Asset Sources

## GDQuest 3D Characters / Sophia

- Source: `https://github.com/gdquest-demos/godot-4-3D-Characters`
- Local copy used for tests: `addons/gdquest_sophia`
- License evidence: `docs/licenses/GDQuest_3D_Characters_LICENSE.txt`
- License: CC-BY 4.0 for art assets, MIT for code/resources generated by Godot.
- Attribution text from the bundled license: `Additional assets CC-By 4.0 GDQuest (https://www.gdquest.com/)`

The copied resource UIDs were stripped from `addons/gdquest_sophia/*.tscn` and `*.tres` files so this project resolves resources by local path instead of stale UIDs from the source project.

## Kenney Animated Characters 3

- Source: `https://opengameart.org/content/animated-characters-3`
- Local copy used for tests: `external/kenney_animated_characters_3/src`
- License evidence: `docs/licenses/Kenney_Animated_Characters_3_LICENSE.txt`
- License: Creative Commons Zero, CC0.

This pack provides the `characterMedium.fbx` skinned model and separate `idle.fbx`, `jump.fbx`, and `run.fbx` animation files used by `tests/test_real_character_assets.gd`.

## Quaternius Ultimate Modular Women Pack

- Source: `https://quaternius.com/packs/ultimatemodularwomen.html`
- Local FBX cache used for conversion: `external/quaternius_modular_women_fbx`
- Local GLB files used for tests: `external/quaternius_modular_women_glb`
- License: Creative Commons Zero, CC0, as listed on the Quaternius pack page.

`Adventurer.fbx` and `Soldier.fbx` were converted to GLB with `tools/convert_fbx_to_glb.py` so the tests can load the real mesh data through Godot's glTF path. `tests/test_real_layered_garment_asset.gd` and `tests/render_real_seam_boundary_smoke.gd` use `Adventurer.glb`.
